Welcome to French Dialogue Builder! 🇫🇷

How to Play:
  1. Choose your level: Beginner, Intermediate, or Advanced
  2. You'll see a dialogue scenario with empty input fields
  3. Type the correct French phrases in the input boxes
  4. Use the 💡 Hint button if you need help
  5. Click 🔊 Listen to hear the correct pronunciation
  6. Click Check Answers to verify your responses
Example Dialogue:

Server: [Input field] ← Type: "Bonjour! Que puis-je faire pour vous?"

You: [Input field] ← Type: "Je voudrais un café, s'il vous plaît"

💡 Hints are available for each line

🔊 Listen to hear correct pronunciation

Features:
  • Score points for correct answers
  • Build streaks for consecutive correct dialogues
  • Track your progress with accuracy rate
  • Use keyboard shortcuts:
    • Enter - Check answers
    • Space - Listen to dialogue
    • → - Next dialogue
Tips:
  • Pay attention to accents and punctuation
  • Use hints if you're stuck (-5 points penalty)
  • Practice pronunciation with the audio feature
  • Try to maintain your streak for higher scores
